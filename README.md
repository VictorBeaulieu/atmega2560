# TOPIC
This a sample to use the amtega2560 from arduino mega board without Arduino IDE.

# FEATURES
- USART
    - init
    - read_char
    - read_string
    - write_char
    - send_string
    - ISR
- I2C
    - i2c_init
    - i2c_start
    - i2c_write
    - i2c_read_ack
    - i2c_read_nack
    - i2c_transmit
    - i2c_receive
    - i2c_writeReg
    - i2c_readReg
    - i2c_stop
